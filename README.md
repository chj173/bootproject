# bootproject
JPA를 이용한 데이터처리페이지

# 개발환경
1. IDE: IntelliJ IDEA Community
2. Spring Boot 3.1.0
3. Java Version - Amazon Corretto version 17.0.7
4. JPA , thymeleaf , Lombok , MySQL8.0.33

# 진행
- 프로젝트 생성<br>
- MySQL 연동(yml)<br>

### 회원가입 및 로그인 
1. 회원가입
2. 로그인
   - session
3. 목록출력
4. 상세조회
5. 수정
6. 삭제
7. 로그아웃
8. ajax통신
   - 이메일 중복확인

### 게시판
로그인(main.html) 게시판표시

1. 글작성
2. 목록
   - 글번호(id)
   - 제목(title)
   - 작성자(writer)
   - 작성일(date)
   - 조회수(hits)
3. 글조회
4. 글수정
5. 글삭제
6. 페이징처리
7. 파일첨부
